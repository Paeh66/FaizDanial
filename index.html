Introduction

Welcome to my Mobile Programming e-Portfolio for Universiti Kebangsaan Malaysia (UKM). This portfolio documents my learning journey throughout the Mobile Programming course, where I developed Android applications using Kotlin and Jetpack Compose. Throughout the semester, I completed five laboratory exercises and two major projects that gradually enhanced my understanding of user interface design, navigation, local databases, cloud integration, APIs, and device sensors.


My main application, CityReport, was developed to support SDG 11: Sustainable Cities and Communities. The application aims to help communities identify and report city-related issues, encouraging a cleaner, safer, and more peaceful living environment. Through this project, I learned how technology can contribute to building happier and more sustainable communities.

Laboratory Activities

Lab 1 - UI

User Interface Design

Introduction to Android Studio and Jetpack Compose for creating modern user interfaces.


Reflection

I learned how to design UI using Android Studio and Jetpack Compose. This lab introduced the foundation of mobile application development and helped me understand layout organization. It improved my confidence in creating visually appealing interfaces.

Lab 2 - Interaction

User Interaction

Implementing interaction between user input and displayed content.


Reflection

I learned how to create interactions between text fields and displayed output using remember state management. This helped me understand reactive programming concepts in Jetpack Compose. It also showed how user actions can dynamically update the interface.

Lab 3 - Material Design

Material Design

Applying Material Design principles to create structured and consistent interfaces.


Reflection

This lab taught me how to create a more structured and professional user interface using Material Design components. I gained experience with themes, cards, buttons, and layouts. The result was a cleaner and more user-friendly application.

Lab 4 - Navigation

Navigation

Building multi-screen applications using navigation components.


Reflection

I learned how to navigate between multiple screens while maintaining application state using mutableState. This lab helped me understand screen routing and user flow. Navigation is an essential component of modern mobile applications.

Lab 5 - Room

Room Database

Implementing local data storage with Room Database, DAO, and entities.


Reflection

I learned how to use Room Database with DAO and database classes to store local application data. This improved my understanding of persistent storage in Android applications. It also prepared me for integrating more advanced data management solutions.
